# Case Study: E-commerce milik Pak Palmer

## Description

Pak Palmer berencana mengembangkan sebuah aplikasi dengan fitur pencatatan produk yang dijual. Aplikasi ini diharapkan dapat mempermudah untuk memantau ketersediaan barang secara real-time sehingga proses pemantauan produk dapat berjalan lancar
Sebagai tahap awal pengembangan, beliau memprioritaskan pada fitur pencatatan stok barang

## What you Need To Do?
- Bersama clan masing-masing, bantulah Pak Palmer untuk melakukan perbaikan dan improvisasi kode program yang telah dibangun (silahkan unduh file ecommerce.java dan product.java di repository ini)
- Minggu depan akan dipilih 3 clan secara acak untuk mempresentasikan hasil pengerjaan di depan kelas. Yang bertugas presentasi adalah anggota clan (dipilih secara acak)
